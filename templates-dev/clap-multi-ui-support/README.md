# clap-multi-ui-support

This is an in-development framework variant which supports native UIs.

It's adding Extensible UI adapter layer to integrate arbitral UI framework or libraries.

## development status

- CLAP targeted (for development easiness)
- tested on MacOS (Apple Silicon)
- `microui` could integrated, but the wrapping layer is still verbose. It's required more easy-to-use wrapper.

## how to build and run

```sh
  cmake --preset ninja
  cmake --build --preset ninja-debug
  sh ./run.sh
```
